## Getting Started

#### Prerequisites

Ensure you have node`12.X.X` or higher

Then

```bash
npm install
npm run prepare
npm run dev
```

Access _localhost_ at http://localhost:8080

#### Debugging localhost

If you have issues try

```bash
npm update
```
